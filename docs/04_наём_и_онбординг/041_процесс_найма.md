# Процесс найма в ОАЭ

## Обзор процесса

```
Вакансия → Поиск → Отбор → Оффер → Документы → Виза → Онбординг
```

---

## Шаг 1: Планирование

### Определение потребности

- Описание должности
- Требования к кандидату
- Бюджет на позицию
- Тип контракта

### Квоты Emiratisation

Для некоторых секторов обязательно соблюдение квот по найму граждан ОАЭ.

---

## Шаг 2: Поиск кандидатов

### Каналы поиска

| Канал | Особенности |
|-------|-------------|
| Job portals | Bayt, LinkedIn, GulfTalent |
| Recruitment agencies | Для специализированных позиций |
| Internal referrals | Рекомендации сотрудников |
| Direct applications | Через сайт компании |

---

## Шаг 3: Отбор

### Типичные этапы

1. Скрининг резюме
2. Телефонное интервью
3. Техническое интервью
4. HR интервью
5. Проверка рекомендаций
6. Background check

---

## Шаг 4: Оффер

После выбора кандидата:

1. Согласование условий
2. Подготовка offer letter
3. Подписание кандидатом
4. Начало визового процесса

---

## Шаг 5: Документы от кандидата

| Документ | Назначение |
|----------|------------|
| Паспорт | Для визы |
| Фото | Для документов |
| Дипломы | Подтверждение квалификации |
| Сертификаты | Профессиональные лицензии |
| Reference letters | Рекомендации |
| Police clearance | Для некоторых позиций |

---

## Шаг 6: Визовый процесс

См. раздел [Рабочая виза](../02_визы_и_разрешения/021_рабочая_виза.md)

---

## Шаг 7: Онбординг

### Первый день

- Встреча с HR
- Знакомство с командой
- Рабочее место и оборудование
- Доступы к системам

### Первая неделя

- Ознакомление с политиками
- Обучение по должности
- Постановка целей

!!! tip "Best Practice"
    Составьте чек-лист онбординга для каждой позиции.
