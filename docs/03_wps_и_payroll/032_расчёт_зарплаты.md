# Расчёт заработной платы

## Структура зарплаты в ОАЭ

Типичная структура compensation package:

| Компонент | Описание |
|-----------|----------|
| Basic Salary | Базовый оклад (основа для расчётов) |
| Housing Allowance | Жилищная надбавка |
| Transport Allowance | Транспортная надбавка |
| Other Allowances | Прочие надбавки |

---

## Базовый оклад (Basic Salary)

Важность basic salary:

- Основа для расчёта end-of-service gratuity
- Основа для расчёта сверхурочных
- Обычно составляет **40-60%** от общей зарплаты

!!! danger "Внимание"
    Занижение basic salary уменьшает выплаты при увольнении. Это законно, но работнику невыгодно.

---

## Минимальная зарплата

В ОАЭ **нет официальной минимальной зарплаты** для частного сектора.

Исключения:

- Некоторые free zones устанавливают минимумы
- Для определённых профессий могут быть требования

---

## Удержания из зарплаты

### Законные удержания

| Тип | Максимум |
|-----|----------|
| Дисциплинарные | 5 дней зарплаты в месяц |
| По решению суда | Согласно решению |
| Погашение займа | По соглашению |

### Запрещённые удержания

- За обучение, предоставленное работодателем
- За рабочую одежду/инструменты
- Депозиты за оборудование

---

## Валюта выплаты

- Стандарт: **AED (дирхамы)**
- Иностранная валюта: по соглашению сторон
- Курс: на дату выплаты

---

```
│         PAYSLIP                 │

├─────────────────────────────────┤

```

---

## Налоги

В ОАЭ **нет подоходного налога** на физических лиц. Зарплата выплачивается без налоговых удержаний.
